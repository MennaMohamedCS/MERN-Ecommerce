# MERN-Ecommerce
# Front-end:
```
HTML
Tailwind CSS
JavaScript
React.js

```
# Back-end:
```
NodeJs
ExpressJs
```
# Database:
```
MongoDB
```
# Handling:
```
CRUD operations
Routing
Authentication & Authorization 
    using:
      -jsonwebtoken
      -bcryptjs
      -jwt-decode  
```
# Features:
```
Sign up / Log in
Add Product
Show Products by Category
Add to cart
Watch List         
History
```

## Project setup
```
npm install
```
## to run Back-end (using all CRUD operations "Rest API ")
```
npm start
```
## to run Front-end project in visual studio code
```
npm start     
```
